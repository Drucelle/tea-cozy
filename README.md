## Overview

This is a fictional responsive tea shop landing page. It was built to match CodeCademy’s provided design specifications.
The project breakdown:
Fixed navigation bar containing Logo on the left and site navigation menu on the top right.
Section 1: Is the tea shop’s Mission section containing a background image and text box covering the image in the middle.
Section 2: Is the Tea Of The Month section with a featured grid. 
Section 3: The Locations section aa background image with the floating text boxes containing the locations’ addresses with a black background.
The Footer section is a column of Info including contact email and phone  number. A copyright text on the bottom left of the page.

## My Process

I analysed the design specification first. Every detail I saw for each section was then written down. As  soon as I felt I had all of the details I needed to work with I then proceeded to start writing the HTML structure first going section by section. Then CSS was added and adjusted the HTML accordingly when needed.
I made sure to stay hydrated and take mini breaks to relax my eyes, strech my legs and moved to get my blood flowing and got back in the zone fully charged again.
